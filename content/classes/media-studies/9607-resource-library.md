# Media Studies · Resource Library

## The coursebook

Marshall, Naisbett & Stoklosa, *Cambridge International AS & A Level Media Studies Coursebook*, CUP (©2016/2023, Cambridge-endorsed, ISBN 978-1-009-26224-8). It has eight units:

| Unit | Book pages |
|---|---|
| 1 Introduction to Media | 2–15 |
| 2 Media Language: Micro elements | 16–58 |
| 3 Media Language: Macro elements | 59–82 |
| 4 Textual Analysis | 83–104 |
| 5 Media Contexts | 105–136 |
| 6 Media Production: AS and A Level portfolios | 137–180 |
| 7 Media Debates | 181–221 |
| 8 Media Ecology | 222–252 |

The book defines terms inline; there is no glossary and no index.

## Past papers

Past papers are available for June 2024 and June 2025, covering Components 2 and 4 only. Components 1 and 3 are coursework.

**Paper 2** (Section A extract analysis and Section B contexts essay, 25 marks each): Section A asks you to analyse how the extract constructs meaning through camera, sound, mise-en-scène, and editing. You get four screenings, and the rubric allows 30/45/45 minutes.

**Paper 4** (A Level): Section A is three 15-mark essays from a fixed three-topic menu in order — regulation, postmodern media, power. Section B is a compulsory Media Ecology synoptic 30-marker.

Command words across all papers: Analyse / Explain / Evaluate / Assess / To what extent.

## How the papers are marked

**Paper 2** (each 25-mark section): five criteria, each marked independently on a 0–5 band, best-fit (a script can sit at different levels on different criteria):

| AO | Criterion | Marks |
|---|---|---|
| AO1 (15) | Media Concepts (language & representation) | /5 |
| AO1 | Contexts & Critical Debates (social significances) | /5 |
| AO1 | Use of Terminology | /5 |
| AO2 (10) | Analysis of how meaning is created, incl. use of theory | /5 |
| AO2 | Use of Examples | /5 |

Per-criterion band words: **5** Sophisticated / Insightful · **4** Thorough / Effective · **3** Clear / appropriate · **2** Limited · **1** Basic · **0** no creditable content. Paper 2 is weighted AO1 60 / AO2 40.

**Paper 4:** Section A's three 15-mark essays each use the same five criteria at 3 marks apiece; Section B's Media Ecology 30-marker uses generic level descriptors (Higher = Levels 4–5, Middle = Level 3, Basic = Levels 1–2). AO1 60 / AO2 40 overall.

This banding is used to mark A1, A4 and the EoT.
